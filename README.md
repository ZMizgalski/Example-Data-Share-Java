## General Information
- This project is a modified version of library back-end and was made for Polish philologists. My task was to create a website where every teacher will be able to add own content and files which will be used by students. Also it has an admin panel.

## Technologies Used
- JWT tokens
- MongoDB database
- Argon2id

## Features
List the ready features here:
- Argon password haser
- Reset password form
- Mail service
- MongoDB database
- Jwt
- @PreAutorized() requests
- Thymeleaf
- Lombok
- Spring security

## Setup
- Gradle
- Java 14
- Mail Config
- MongoDB database

## Usage
1. Create a folder named you like.
2. Inside your folder, open console and paste: `git clone https://github.com/ZMizgalski/Example-Data-Share-Spring-Boot-Java.git`
3. Then you are ready to go just open it in any Editor. (I prefer Intellij Idea Community) [Download here](https://www.jetbrains.com/idea/download/#section=windows)
